# Chatbot AI

- Giai đoạn 1 – Phân tích và thiết kế (Tuần 1)
+ Xác định mục tiêu, phạm vi chatbot.
+ Tìm hiểu mô hình chatbot.
+ Thiết kế sơ đồ hoạt động và giao diện cơ bản.

- Giai đoạn 2 – Xử lý dữ liệu & Huấn luyện mô hình (Tuần 2)
+ Tiền xử lý dữ liệu hội thoại (loại bỏ ký tự thừa, chuẩn hóa chữ).
+ Xây dựng mô hình trả lời.
+ Kiểm thử mô hình bằng vài câu hỏi thử.

- Giai đoạn 3 – Xây dựng giao diện chatbot (Tuần 3)
+ Thiết kế giao diện chatbot (web hoặc terminal).
+ Kết nối mô hình trả lời vào giao diện.
+ Kiểm thử chatbot toàn hệ thống.

- Giai đoạn 4 – Hoàn thiện và viết báo cáo (Tuần 4)
+ Hoàn thiện chatbot, chỉnh sửa lỗi.
+ Đánh giá độ chính xác, tốc độ phản hồi.
+ Viết báo cáo, trình bày kết quả, vẽ biểu đồ Gantt.
